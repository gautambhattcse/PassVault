# PassVault

## Overview
PassVault is a user-friendly password manager that helps you securely store and manage your passwords along with website URLs and usernames. Built using Vite, ReactJS, and TailwindCSS, PassVault ensures a seamless and efficient user experience.

## Features
- **Secure Storage:** Safely saves passwords, website URLs, and usernames in local storage.
- **Easy to Use:** Simple and intuitive interface for managing your passwords.
- **Fast Performance:** Built with Vite for optimized and fast build tooling.
- **Responsive Design:** Styled with TailwindCSS for a modern and responsive look.

## Technologies Used
- **Vite**: For fast and optimized build tooling.
- **ReactJS**: To create a dynamic and responsive user interface.
- **TailwindCSS**: For modern and efficient styling.

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/passvault.git
   ```
2. Navigate to the project directory:
   ```sh
   cd passvault
   ```
3. Install the dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```

## Usage
1. Open the application in your browser (usually at `http://localhost:3000`).
2. Add a new password entry by entering the website URL, username, and password.
3. Manage your passwords through the user-friendly interface.
